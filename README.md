## Проект: Сервис бронирования переговорок

### Как запустить
```bash
docker-compose up --build
```
Сервис будет доступен на `http://localhost:8080`.

### Принятые технические решения (MVP)
В рамках тестового задания и ограниченного времени, основные усилия были направлены на **бизнес-логику** и соблюдение **контракта API**.

**In-Memory хранилище:** Для текущей версии выбрано хранение данных в оперативной памяти. Это позволило гарантировать экстремально низкий пинг и сосредоточиться на логике фильтрации слотов и бронирований без трат времени на настройку миграций БД.
Код хендлеров отделен от моделей, что позволяет быстро заменить слайсы на PostgreSQL.
